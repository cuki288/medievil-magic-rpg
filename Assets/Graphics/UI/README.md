# UI Directory

This directory contains user interface elements for Medievil Magic RPG.

## UI Elements
- MainMenu.png - Main menu background
- InventoryPanel.png - Inventory panel design
- CharacterPanel.png - Character status panel
- QuestPanel.png - Quest display panel
- SkillPanel.png - Skill selection panel

## Buttons
- Button_Normal.png - Normal button state
- Button_Hover.png - Hover button state
- Button_Click.png - Clicked button state

## Guidelines
- All UI elements should maintain consistent design language
- Use appropriate resolution (1080p recommended)
- Maintain transparency where needed
- Organize by UI component type
