# Graphics Directory

This directory contains all visual assets for the Medievil Magic RPG game.

## Structure
- Characters/ - Character sprites and models
- Environments/ - Backgrounds and environment assets
- UI/ - User interface elements
- Effects/ - Particle effects and animations
- Items/ - Item sprites and icons

## Guidelines
- All graphics should follow the game's visual style
- Resolution should be consistent across all assets
- Use appropriate file formats (PNG for sprites, JPG for backgrounds)
