# Environment Directory

This directory contains background images and environment assets for Medievil Magic RPG.

## Backgrounds
- Overworld.png - Main overworld background
- Dungeon1.png - Dungeon level 1 background
- Castle.png - Castle environment background
- Forest.png - Forest environment background

## Tilesets
- Tileset_Environment.png - Environment tileset for map building
- Tileset_Objects.png - Object tileset for decorations

## Guidelines
- All backgrounds should be 1920x1080 resolution
- Maintain consistent art style
- Use appropriate file formats (PNG for transparency)
- Organize by environment type
