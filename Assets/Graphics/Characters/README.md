# Characters Directory

This directory contains character sprites and models for Medievil Magic RPG.

## Sprite Sheets
- PlayerCharacter.png - Main player character sprite sheet
- EnemyCharacter.png - Enemy sprite sheet
- NPCCharacters.png - Non-player character sprite sheets

## Guidelines
- All sprites should be 32x32 pixels
- Use consistent color palette
- Maintain sprite sheet organization
- Save as PNG format with transparency
