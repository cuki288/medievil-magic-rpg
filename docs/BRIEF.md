# BRIEF.md - Medievil Magic RPG

## Problem
Create a medieval fantasy RPG game with character progression, combat systems, quests, and inventory management using Unity.

## Goals
- Implement a complete RPG experience with character progression
- Develop robust combat mechanics with turn-based system
- Create quest and story system
- Build comprehensive inventory and equipment management

## Scope
- Character system with classes (Warrior, Mage, Rogue, Cleric)
- Turn-based combat mechanics
- Inventory and equipment system
- UI components for character, skills, and inventory
- Quest tracking system

## Success Criteria
- Game runs without major issues
- Character progression system works correctly
- Combat system provides engaging gameplay
- UI elements are clearly displayed and functional
- All core RPG features implemented
